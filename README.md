# EcoRoot - Sustainable Bamboo Products

EcoRoot is a web application designed to promote sustainable bamboo products, allowing users to upload products, sign up, and log in to manage their profiles. The application supports user authentication and file uploads for product images.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Installation](#installation)


## Features

- User Signup and Login
- Product Image Upload
- PostgreSQL Database Integration
- RESTful API for product management
- Responsive Design

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Authentication:** bcrypt.js for password hashing
- **File Uploads:** multer for handling multipart/form-data
